## Overview

This project explores **Netflix content trends** from 2008 to 2021, analyzing **Movies vs TV Shows**, genres, countries, top directors, and content durations.

The dataset contains **7,789 titles** with metadata including category, director, cast, country, release date, rating, duration, and genre.

## Key Columns

| Column       | Description                  |
| ------------ | ---------------------------- |
| Category     | Movie or TV Show             |
| Title        | Name of the content          |
| Director     | Director(s)                  |
| Country      | Country of production        |
| Release_Date | Netflix release date         |
| Rating       | Content rating               |
| Duration     | Runtime (minutes) or seasons |
| Type         | Genre(s)                     |


## Analysis Highlights

* **Trends over time:** Movies dominate, but TV shows grew steadily after 2015
* **Top genres:** International Movies, Dramas, Comedies, International TV Shows, Documentaries, Action & Adventure
* **Country contributions:** U.S., India, U.K., Canada, France, Japan lead content production
* **Top directors:** Most featured directors identified
